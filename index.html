Why Tabula exists.

Tabula was built out of a real and daily frustration. Working across multiple complex projects simultaneously — each with its own research threads, reference materials, active documents, and in-progress work — the browser tab bar became unmanageable fast.

Native Chrome tab groups helped at the surface level. But the real problem was deeper. There was no way to see the full shape of what was being worked on. No way to pick up a thread exactly where it was left off. No way to switch between projects without losing mental context in the process.

What was needed was not a better list of tabs. What was needed was a desk. A physical desk gives you spatial memory — you know where things are because you put them there. You can see what is active, what is waiting, and what can be set aside. The browser offered none of that. Tabula was built to give it back.

Everything on your desk, finally organized.

Every feature in Tabula was designed around one question: what would a real desk do?

🗂

Workspaces

Named, color-coded drawers for every project, context, or topic. Tabs live exactly where you put them and stay there across restarts.

🖥

Canvas View

Arrange tab cards freely on a 2D surface. Position things spatially — front and center for what is active, pushed aside for what is waiting.

📝

Sticky Notes

Every tab card holds a note that follows the URL everywhere. Come back three days later and know instantly why you opened it.

🧹

Sweep

One keyboard shortcut clears all loose tabs into a named workspace in under half a second. The fastest path from overwhelmed to organized.

🔍

Command Bar

Type plain English to control your entire tab workspace. "Show old tabs." "Create a Research workspace." "Group by domain." It just works.

🤖

On-Device AI

Tab suggestions, auto-summaries, and semantic relationship mapping — all running locally in your browser. No account. No API key. No data sent anywhere.

☁️

Zero-Account Sync

Workspaces and notes sync across every Chrome instance signed into your Google account. No backend. No subscription. Your data stays yours.

💾

Session Save

Save any workspace as a snapshot and restore it later — every tab reopened exactly as it was. Set work down completely and pick it back up perfectly.

A desk, not a list.

Every other tab manager organizes tabs as a list. Tabula starts from a different premise entirely.

1

Spatial, not hierarchical

Lists require you to remember where you put things. A desk lets you see where things are. Tabula gives tabs positions, relationships, and context — not just names in a tree.

2

AI that respects your attention

Tabula's AI never makes decisions for you. It notices patterns, surfaces suggestions, and stays quiet unless it has something worth saying. All of it runs on your device with no external service.

3

No infrastructure required

Tabula has no backend, no database, no server, and no subscription. It is a Chrome Extension with zero ongoing costs beyond the Chrome sync API that Google provides for free.

4

Built for how people actually work

Not for browsing one website at a time. For managing complex, simultaneous, multi-thread work across dozens of tabs every single day.

Try it in two minutes.

Tabula installs directly from the source code — no Chrome Web Store required.

1

Download the extension

Click the GitHub button above, then download the repository as a ZIP file using the green Code button.

2

Open Chrome Extensions

In Chrome, navigate to chrome://extensions and turn on Developer mode in the top right corner.

3

Load the extension

Click Load unpacked and select the Tabula folder you downloaded. The extension appears immediately.

4

Open your desk

Click the Tabula icon in your Chrome toolbar. Your tab desk opens in the side panel. Start organizing.
